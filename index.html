Andrew WR Campbell is a humanoid Front-End Web Developer and Multi-disciplinary Designer.

I am dedicated to creating smart, accessible and responsive web applications, informed by my history as a lead graphic designer. I'm always excited to bring my analytical mindset and problem-solving creativity to an energetic and highly collaborative environment.

Progress is the driving force behind everything I do; societal, technological, ecological and personal. With that in mind, I follow the guiding principle of "functional minimalism", a practice of starting with what is necessary for user experience, and building out from there. With varied influences, from spacious electronic music to technical clothing architecture, I combine this ethos and those inspirations into the focus of my vision.

MTG Gateway

A vanilla JavaScript based application that introduces users to the basic concept of the Magic: the Gathering tabletop card game. Uses an API to call a random card from each of the six basic types of cards in the game, and dynamically changes the descriptions based on the call.

Veilance Archive

The Veilance Archive is an application that allows users to to view images and descriptions of previous seasons of the clothing brand Veilance, which is unavailable otherwise. The database is managed using Firebase to make calls for the specific season based on user requests.

WriteOn

WriteOn is a React based journaling application that encourages users to write utilizing a timer and prompt. Uses Firebase to manage the daily posting of prompts, including those submitted by users.
